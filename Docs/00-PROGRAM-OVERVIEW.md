## Technical Summary

| Component | Technology | Purpose |
|-----------|------------|---------|
| Interface | NiceGUI (Python) | Modern web-style UI in browser |
| Database | SQLite | Stores card inventory locally |
| Card Scanning | OpenRouter API | AI vision to read card photos |
| Image Hosting | ImgBB API | Provides public URLs for Whatnot |
| Price Research | Terapeak (browser) | Opens pre-filled searches |
| Export | CSV generation | Whatnot bulk upload format |

**Runs locally:** All data stays on your computer. Nothing stored in the cloud except:
- Card images sent to OpenRouter for scanning (not stored)
- Images uploaded to ImgBB for Whatnot listings

**User provides credentials:** You create your own API accounts and enter the keys. Card Lister never sees your passwords.

## Limitations (Honest Assessment)

**AI isn't perfect:**
- Obscure parallels may not be identified correctly
- Damaged/worn cards harder to read
- Always review AI results before saving

**Manual steps still required:**
- Taking photos (can't automate that)
- Reviewing Terapeak for pricing (no API available)
- Final review in Whatnot before publishing

**Requires internet:**
- For card scanning (OpenRouter)
- For image upload (ImgBB)
- For price research (Terapeak)
- Local database works offline, but can't scan/upload

**Learning curve:**
- First-time setup takes ~10 minutes
- Need to create OpenRouter and ImgBB accounts
- But after setup, it's simple
