## Card Types

### Base Card
Standard card from a set with no special features.

### Parallel
Same card as base but with different coloring, border, or finish.
- **Numbered:** Has print run (e.g., /99, /25, /10, 1/1)
- **Unnumbered:** No print run stated but visually different

### Insert
Special subset within a product, often with unique design.
- Examples: Rated Rookie, Downtown, Kaboom!, Color Blast

### Short Print (SP)
Intentionally printed in lower quantities than base cards.

### Super Short Print (SSP)
Even rarer than SP.

### Autograph (Auto)
Contains player's actual signature (ink or sticker).

### Relic / Memorabilia
Contains piece of game-used or event-worn material (jersey, patch, etc.).

### Rookie Card (RC)
Player's first licensed card (designated with RC logo or "Rookie" text).

## Identifying Features

### Rookie Card Indicators
- "RC" logo/badge
- "Rated Rookie" (Donruss)
- "Rookie" text
- First-year player in their first licensed set

### Serial Number Location
Usually bottom-left or bottom-center of card front.
Format: `045/199` (card 45 of 199 total)

### Autograph Types
- **On-card:** Signed directly on card
- **Sticker:** Signed sticker affixed to card
- **Cut:** Historical signature cut from other source

### Relic Types
- **Single swatch:** One window with material
- **Multi-swatch:** Multiple windows
- **Patch:** Includes part of jersey number/logo (more valuable)
- **Prime:** Premium material (limited)

## Condition Grading (Raw Cards)

| Grade | Description |
|-------|-------------|
| Gem Mint | Perfect, PSA 10 candidate |
| Mint | Near-perfect, PSA 9 candidate |
| Near Mint (NM) | Minor imperfection visible under close inspection |
| Excellent (EX) | Light wear on corners/edges |
| Very Good (VG) | Moderate wear, minor creasing |
| Good | Heavy wear, creasing, possible damage |
| Poor | Major damage |

### What to Look For
- **Centering:** Left-right and top-bottom alignment
- **Corners:** Sharpness, dinging, whitening
- **Edges:** Clean vs. chipping
- **Surface:** Scratches, print lines, staining

## Quick ID Checklist for Scanning

When analyzing a card image, identify:

1. ☐ Player name (front of card)
2. ☐ Card number (usually bottom or back)
3. ☐ Year (may need to infer from set/design)
4. ☐ Manufacturer logo
5. ☐ Brand/set name
6. ☐ Team (uniform, logo, or text)
7. ☐ Parallel color/pattern
8. ☐ Serial number (if present)
9. ☐ Rookie indicator (RC badge)
10. ☐ Auto (signature visible)
11. ☐ Relic (swatch window visible)
